# 介绍


- 三斜线指令是包含单个 **XML标签** 的单行注释。注释内容会被作为编译器指令使用。

- 三斜线指令只可放在包含它的文件的最顶端。并且一个三斜线指令的前面只能出现单行或多行注释，当然也包括其他的三斜线指令。如果它们出现在一个语句或声明之后，那么它们会被当做普通的单行注释，并且不具备任何含义。
