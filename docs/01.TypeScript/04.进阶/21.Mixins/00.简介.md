# 介绍



除了传统的 OO 层次结构，从可重用组件构建类的另一种流行方法是通过组合更简单的部分类来构建类。您可能熟悉 **Scala** 等语言的 `mixins` 或 `trait` 的概念，并且该模式在 JavaScript 社区中也很流行。
