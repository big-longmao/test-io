# 基础

在 TypeScript 中，如果我们在定义变量时并没有显示为其指定类型，类型推断会帮助我们提供类型。如下所示：

// let x: number
let x = 3;
1
2

x 变量的类型被推断为 number。这种推断发生在初始化变量值和成员值、或设置函数参数默认值和确定函数返回类型时。

在大多数情况下，类型推断是直接的。在下面的小节中，我们将探讨如何推断类型的一些细微差别。
